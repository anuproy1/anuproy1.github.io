# Anup Kumar Roy Portfolio Website

## How to use
1. Create a GitHub repository named `YOUR-GITHUB-USERNAME.github.io`.
2. Upload all files from this folder.
3. Open `index.html` and replace:
   - `YOUR-GITHUB-USERNAME`
   - `YOUR-LINKEDIN`
   - `YOUR-GOOGLE-SCHOLAR-LINK`
4. Go to repository Settings > Pages.
5. Select Deploy from branch > main > root.
6. Open `https://YOUR-GITHUB-USERNAME.github.io`.

## How to add daily posts
Edit `posts.json` and add a new item:

```json
{
  "title": "Day 4: My New VLSI Note",
  "date": "2026-06-17",
  "category": "VLSI",
  "summary": "Short description here.",
  "link": "https://github.com/YOUR-GITHUB-USERNAME/repository-name"
}
```
